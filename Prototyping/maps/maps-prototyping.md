````md
# AgroPulse Maps

AgroPulse Maps is the geospatial intelligence module of the AgroPulse platform.

This application uses Copernicus Sentinel satellite data, geospatial analytics, and AI-driven insights to help large-scale agricultural producers monitor productivity, optimize irrigation, and predict climate risks.

## Features

- Smart Yield Heatmaps
- NDVI Analysis
- Climate Monitoring
- Predictive Productivity Zones
- Irrigation Prioritization
- Real-Time Agricultural Dashboard

## Technologies

- Next.js
- React
- TypeScript
- Tailwind CSS
- Mapbox
- Leaflet
- Copernicus Sentinel Data
- GeoJSON

## Setup

```bash
npm install
npm run dev
````
