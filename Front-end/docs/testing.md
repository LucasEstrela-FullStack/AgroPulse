# Testing

## Tooling

- Vitest
- React Testing Library
- Playwright

## Coverage target

- aim for 80 percent or higher coverage

## What to test

- shared components
- feature flows
- state management
- service adapters
- map interactions
- form validation
- page rendering

## Quality goals

- keep unit tests close to the logic they protect
- use integration tests for key user journeys
- use end-to-end tests for critical product flows
